不只是 AI

一个完整、好用的原生 SSH 客户端。

  • 内置真实终端(PTY)，多标签、⌘1–9 切换、密码/密钥自动登录
  • 全局快捷连接 ⌘K，模糊搜索别名/描述/分组，回车即连
  • 可视化 SFTP 双栏拖拽传输，右键 AI 改文件
  • 图形化端口转发(ssh -L，支持跳板机，一键启停)
  • 状态栏实时 远端 CPU / 内存 / 时间
  • 命令片段库 多行命令、{{占位符}} 运行时填参
  • 整窗壁纸 图片或循环 mp4，面板半透明叠加
  • 多款配色 Dracula / Nord / Tokyo Night… 或导入 iTerm2
  • 可自定义快捷键 录制式设置、冲突检测
  • 加密备份 到文件或 WebDAV，一键迁移
  • 后台巡检 + 掉线通知，AI 自动归因
  • 中文 / English 运行时切换

安全与隐私

无服务端、无中心数据库，所有状态都在本机。

🔑

密码只进 Keychain

主机密码与 AI Key 仅存 macOS 钥匙串，配置文件不落明文。

🧾

标准配置单一来源

始终读写标准 ~/.ssh/config；卸载后 ssh/scp/VS Code Remote 照常可用。

🛡

AI 可控

发送前脱敏；生成命令永不自动执行；高危命令强制确认，可开只读沙盒。
