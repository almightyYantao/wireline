舰队 · Fleet

单机 AI 终端给不了的能力:一次操作一整个分组——因为 Wireline 掌握你的主机图谱。

一句话，多主机

选中若干主机或一整个分组，描述一次任务，并发跑遍全部。

是汇总，不是刷屏

逐机输出可折叠，顶部由 AI 提炼成一个结论或一张对比表。

无需先连接

非交互引擎直接跑批，不必逐台开会话。

规模化也安全

高危命令在触及舰队前仍强制二次确认；发送前脱敏。

认得分组与别名

直接用 ~/.ssh/config 里的别名或分组名指定目标——和 ssh 同一套名字。

你的模型跑舰队

汇总用你自己的 endpoint 或完全本地的 Ollama，敏感运维不出本机。

Wireline vs Termius vs Warp

对手强的格子如实标出；真正没人同时具备的,是最后三条的交集。

WirelineTermiusWarp
定位原生 macOS SSH 舰队管理器跨平台 SSH 客户端AI 原生终端
主机数据存在标准 ~/.ssh/config✅ 唯一源❌ 自家 vault— 不管理主机

Warp 加了本地 AI,却仍不管你的主机;Termius 管主机,却把它们锁进自家 vault、AI 只跑在它的云上。Wireline 把主机留在标准 ~/.ssh/config、AI 跑在你自己的模型上(或完全离线)、一句话指挥整支舰队。

竞品能力为 2026-07 时点信息 —— 迭代很快,以最新版本为准。

AI 是杠杆

AI 让上面这一切毫不费力,不是外挂功能。接 OpenAI 兼容服务(中转站/直连)或本地 Ollama;数据可脱敏、命令永不自动执行。

自然语言 → 命令

说需求，生成命令，[插入] 或 [运行]；命令栏 ⌘; 更快。

Agent 自动执行

给命令 → 真实执行 → 拿输出 → 继续，直到出结论。可见/旁路、危险确认、只读沙盒。

舰队群跑

多选主机 → 一句话 → 并发执行 → AI 汇总成结论/表格。单终端做不到。

诊断 · 解释 · 总结

一键分析报错、解释命令(高危警告)、总结长日志。

主机记忆

记住每台机器的稳定信息，之后回答自动结合，越用越懂你的环境。

AI 副驾操作客户端

一句话建隧道、加主机、连接、开文件、跑片段——均需确认。

MCP 工具（本地优先）

接 stdio / HTTP 的 MCP server（filesystem / GitHub / k8s…），你的模型直接调用；密钥进 Keychain，写操作二次确认。

内置运维技能

负载/磁盘/服务/容器/安全等排查流程，AI 按需载入、多为只读；可自定义。

SFTP AI 改文件 · 变更评审

右键远程文件说改动 → 预览 → 影响/风险评审 → 写回。

会话复盘 → Runbook

把本次操作整理成带步骤/命令/回滚的 Markdown 手册。

桌面宠物

悬浮桌面的小精灵，独立对话窗口。⌥⌘J 全局唤出(任意 App 都行)，说「把 IAI 所有机器的 Docker 状态总结下」，它按分组并发执行再汇总。

不只是 AI

一个完整、好用的原生 SSH 客户端。

  • 桌面宠物 悬浮小精灵、⌥⌘J 全局唤出(任意 App)，自然语言按机器/分组执行并汇总

  • 状态栏实时 远端 CPU / 内存 / 时间

  • 多款配色 Dracula / Nord / Tokyo Night… 或导入 iTerm2
  • 可自定义快捷键 录制式设置、冲突检测
  • 加密备份 到文件或 WebDAV，一键迁移

  • 中文 / English 运行时切换

安全与隐私

无服务端、无中心数据库，所有状态都在本机。

密码只进 Keychain

主机密码与 AI Key 仅存 macOS 钥匙串，配置文件不落明文。

标准配置单一来源

始终读写标准 ~/.ssh/config；卸载后 ssh/scp/VS Code Remote 照常可用。

AI 可控

发送前脱敏；生成命令永不自动执行；高危命令强制确认，可开只读沙盒。
